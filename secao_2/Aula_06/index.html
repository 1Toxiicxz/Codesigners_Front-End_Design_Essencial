O que é CSS?

Linguagem de Formatação em Folhas de Estilos

Cascading Style Sheets (CSS) é uma linguagem de folhas de estilo utilizada para definir a apresentação visual de documentos escritos em uma linguagem de marcação, como HMTL ou XML. Com ele você pode atribuir cores, família de fontes, tamanhos e muitas outras formatações às tags HTML.

CSS é lindo!

As funcionalidades que todo designer queria!

CSS3 é a versão mais recente do CSS, onde suas principais mudanças foram a possibilidade de criar efeitos de transição, manipular elementos tridimensionalmente, permitir efeitos de sombras, bordas arredondadas, media queries e muito mais!
