TAGS!

Sintaxe: Tags HTML são escritas da seguinte forma = <abretag> conteúdo </fechatag>

Semântica - Qual TAG eu uso e pra quê?

Cada componente visual em nossa interface tem um tipo de tag HTML mais adequado para sua contrução. Todavia, essa não é a maneira mais correta de se fazer isso, uma vez que existe hierarquia correta de Tags a ser utilizada.
Com a chegada do HTML 5 ganhamos mais tags que ajudam os navegadores e dispositivos móveis a identificarem melhor os componentes de interface.

Toda a documentação do HTML pode ser encontrada no site w3schools.com https://www.w3schools.com/tags/default.asp
