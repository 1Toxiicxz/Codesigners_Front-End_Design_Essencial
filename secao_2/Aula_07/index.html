Classes

Atribuímos um nome à nossa classe, informamos as propriedades de formatação CSS, e depois aplicamos o atributo 'class' à tag HTML que queremos formatar. 

    
.txt-preto {
  font-family: Arial;
  font-size: 32px;
  color: #000;
}

.txt-azul {
  font-family: Arial;
  font-size: 18px;
  color: #3333ff;
}
