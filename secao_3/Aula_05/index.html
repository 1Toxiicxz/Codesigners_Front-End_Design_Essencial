O que são GRIDS?

Estruturação de Layouts de Páginas Web

As Grids não são propriamente uma obrigação para o desenvolvimento de sites, mas utilizar um sistema de grid ajuda a manter coerência de alinhamento dos elementos das páginas, organizar eos elementos de interface (User Interface - UI), além de facilitar o desenvolvimento do front-end e ajudar outros webdesigners que possam por ventura vir a trabalhar no mesmo projeto posteriormente.

Divisão de Grid em Colunas

12 Colunas - por padrão os frameworks trazem seus sistemas de grid dividido em 12 colunas, mas caso você precise é possível dividir em mais colunas editando o código CSS do framework escolhido;
